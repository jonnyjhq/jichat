# J信

#### 介绍
J信是一款基于原生Android的IM应用，主要面向企业用户，包含了通讯录，群聊，单聊，朋友圈，公众号，小程序等一系列完整的业务功能，遵循Google Material Design UI设计规范，可能是微信之外体验最好的即时通讯App，此版本目前只提供了Android客户端。

#### 丰富的消息通知
消息列表可以显示各种消息通知一条不落 并且可以置顶和标记阅读状态
